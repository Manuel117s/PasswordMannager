La aplicacion constara de lo siguiente:

1.- Un guardado de contraseñas con SQLite
2.- Encryptacion de estas contraseñas por medio de una contraseña maestra 
3.- La base de datos queda encriptada hasta que se de la contraseña maestra

Ejemplo de SQLite:

import sqlite3
from pathlib import Path

conection = sqlite3.connect("tutorial.db")
cursor = conection.cursor()

dataBaseFile = Path("tutorial.db")

cursor.execute("CREATE TABLE IF NOT EXISTS movie(tittle, year, score)")

cursor.execute("""
    INSERT INTO movie VALUES
    ('Monthy python and the holy grail', 1975, 8.2),
    ('Sophie´s 6 hour disk drive odysey', 2026, 9.0)
""")

for row in cursor.execute("SELECT tittle, year, score FROM movie"):
    print(row)

conection.close()
